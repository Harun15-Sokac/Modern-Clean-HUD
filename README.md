# 🚀 Advanced FiveM HUD

A modern, clean, and optimized HUD for FiveM servers.
Designed to provide all essential player information without clutter, focusing on smooth UX and performance.

---

## ✨ Features

* 🍔 **Food & Water System**

  * Real-time hunger and thirst display
  * Smooth updates and animations

* 💰 **Money Display**

  * Cash and bank balance
  * Automatic updates

* 🚗 **Car HUD**

  * Speed (km/h)
  * Fuel level


* 🔫 **Weapon HUD**

  * Ammo counter
  * Weapon icon
  * Reload indicator

* 🧠 **Optimized**

  * 0.00ms idle usage
  * Lightweight NUI
  * Clean and readable code

## 📦 Installation

1. Download the repository:

```
git clone https://github.com/Harun15-Sokac/Modern-Clean-HUD
```

2. Place it in your `resources` folder

3. Add to your `server.cfg`:

```
ensure your-hud
```

4. Restart your server
